13 / 4 RÉSEAUX DÉTAILLÉS

Chaque API en détail.

f

Meta Graph API v19+

Auth
OAuth2 → short-lived token → exchange for long-lived (60j) → page token (permanent). Sélection de la Page à publier.
Publish
POST /{page-id}/photos (image) ou /{page-id}/feed (lien). Upload multipart. Réponse : post_id pour tracking.
Metrics
GET /{post-id}/insights (post_impressions, post_reactions_by_type_total, post_clicks). Disponibles 24h après publication.
⚙

Instagram Graph API

Auth
Même token Meta. Scope supplémentaire : instagram_content_publish. IG Business Account lié à la Page Facebook.
Publish
2 étapes : POST /{ig-user}/media (crée container avec image_url) puis POST /{ig-user}/media_publish (publie). Image obligatoire.
Metrics
GET /{media-id}/insights (impressions, reach, likes, comments, saved). Pas de "shares" sur IG.
in

LinkedIn Marketing API v2

Auth
OAuth2 séparé de Meta. Scopes : w_member_social, r_organization_social. Token 60j, refresh disponible.
Publish
3 étapes : registerUpload → PUT binary → POST ugcPosts. Texte : max 3000 chars. Format pro forcé.
Metrics
GET /organizationalEntityShareStatistics (impressionCount, clickCount, likeCount, commentCount). Délai 24-48h.
G

Google Business Profile API

Auth
OAuth2 Google. Scope : business.manage. Réutilisation du token Advice si déjà connecté.
Publish
POST /accounts/{id}/locations/{id}/localPosts. Image + texte + CTA (LEARN_MORE, CALL, etc). Posts expirent après 7j.
Metrics
GET /locations/{id}/localPosts/{id}/insights (viewCount). Métriques limitées mais visibilité locale forte.

20 / EDGE CASES

Quand ça ne se passe pas bien.

Réseau down en pleine publication

Les 3 autres publient normalement. Le post en échec passe en file d'attente. Retry auto. Notification : "Facebook publié avec 15 min de retard".

Upload image échoue

Retry 3x. Si échec total : publication text-only (sauf Instagram qui exige une image → reporté + notification au courtier).

LinkedIn rate limit (100 posts/jour)

Compteur par app. Staggering des publications. Si limite atteinte : file d'attente + publication au prochain créneau libre.

Token révoqué par le courtier

Détection immédiate. Désactivation propre du réseau. Badge rouge dans le dashboard. Invitation à reconnecter en 1 clic.

Meta change son API

Version pinée (v19.0+). Couche d'abstraction : si Meta change un endpoint, on ne modifie qu'un seul fichier.
