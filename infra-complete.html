08 / CE QUI EST INCLUS (1/3)

Setup & sécurité.

1

Setup serveur Hetzner

Je commande le VPS, j'installe Ubuntu 24, et je sécurise tout.

Installation Ubuntu 24
Clé SSH (pas de mot de passe)
Firewall (ufw) configuré

En clair : le serveur est installé, verrouillé, et seules les bonnes portes sont ouvertes (web + SSH).

2

Nginx — l'aiguilleur du serveur

Nginx fait l'aiguillage. C'est lui qui décide où envoyer chaque visiteur.

Quelqu'un va sur... app.brokerone.insure → Nginx envoie vers Advice (port 3001) brokerone.insure → Nginx envoie vers Site vitrine (port 80) n8n.brokerone.insure → Nginx envoie vers N8N (port 5678) ← futur

En clair : un seul serveur, plusieurs adresses. Nginx fait le tri automatiquement.

09 / CE QUI EST INCLUS (2/3)

SSL, DNS, site vitrine.

3

SSL / HTTPS — le cadenas vert

Certificats Let's Encrypt gratuits, renouvelés automatiquement. Toutes les connexions sont chiffrées.

🔒 https://app.brokerone.insure — connexion sécurisée

En clair : le petit cadenas vert dans le navigateur. Gratuit et automatique.

4

DNS / Sous-domaines

Je configure les adresses pour que tout pointe vers le bon endroit.

A brokerone.insure → site vitrine
A app.brokerone.insure → application Advice

Note : je te fournis les IPs et les instructions. Tu devras les entrer dans ton registrar (OVH, Gandi, etc.).

5

Migration site vitrine

Le site brokerone.insure (actuellement hébergé ailleurs) migré sur le même serveur. Un seul endroit pour tout gérer.

10 / CE QUI EST INCLUS (3/3)

Fiabilité & livraison.

6

PM2 — le gardien de l'application

PM2 garde l'application vivante 24h/24. Si l'app plante (bug, erreur mémoire...), PM2 la redémarre automatiquement en 2 secondes.

PM2 surveille en permanence : Advice → online (si crash → redémarre auto) Sentinel → futur Connect → futur

En clair : même à 3h du matin, si l'app plante, elle revient toute seule.

7

Monitoring basique

Vérification automatique que le serveur est en ligne. Alerte email si quelque chose tombe.

8

Livraison à Bruno

Documentation de l'architecture
