05 / PAGE TAMPON

Routage intelligent.

MVP

Le client clique le lien dans l'email et arrive sur une page brandée aux couleurs du courtier (logo + couleurs). Il note de 1 à 5 étoiles et laisse un commentaire.

Note ≥ 4 étoiles

Redirigé vers la fiche Google du courtier pour laisser un vrai avis public.

Note < 4 étoiles

Avis capturé en interne. Le courtier reçoit une alerte. Le client insatisfait ne finit pas sur Google.

Résultat : les avis positifs vont sur Google, les négatifs restent privés. Le courtier maîtrise son e-réputation.

PARTIE 7 — PRODUCTION
PROD

Edge Cases.

Le MVP gère le happy path. La production gère tout le reste.

PROD

Token OAuth expiré pendant une campagne

L'envoi d'email échoue silencieusement. Fix : Vérification token avant chaque envoi. Si expiré, refresh auto. Si refresh échoue, notification au courtier + pause campagne.

PROD

Apify quota dépassé (scraping Google)

Le cron de 6h échoue, plus de sync Google. Fix : Alerte admin + retry à 12h. Compteur de quota visible dans les paramètres.

PROD

Claude API timeout sur l'analyse d'avis

Fix : Retry 3x avec backoff exponentiel. Si échec total, avis marqué "analyse en attente" + bouton relancer.

PROD

Import CSV avec données malformées

Emails invalides, colonnes manquantes, encodage foireux. Fix : Validation ligne par ligne. Rapport d'import avec lignes acceptées/rejetées.

PROD

Courtier supprime son compte Google

La connexion Google Business tombe. Fix : Détection auto, notification, désactivation propre, invitation à reconnecter.

PROD

2 courtiers avec le même SIREN/BCE

Fix : Contrainte unique en base. Gestion du conflit avec message clair : "Ce numéro est déjà utilisé".
